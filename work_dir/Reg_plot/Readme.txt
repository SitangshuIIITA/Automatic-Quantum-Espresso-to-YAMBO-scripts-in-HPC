1. This requires python-matplotlib library in your local system.
2. The python script is used to perform a quick regression plot and cross-verification of GW energy data. 
3. Once the GW calculation is done in your HPC, you will see the following text files: GW_positiveE.txt, GW_negativeE.txt, slope_c.txt slope_v.txt starts to appear in this folder.
4. Copy this folder now in your local system. 
5. Hit python plt_reg.py from your local terminal.
Enjoy:)
