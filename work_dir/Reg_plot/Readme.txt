1. This requires python-matplotlib library.
2. The python script is used to perform a quick regression plot and cross-verification of GW energy data. 
3. Run the script when GW calculation is done. You will see the following text files: GW_positiveE.txt, GW_negativeE.txt, slope_c.txt slope_v.txt will appear in this folder.
4. Copy this folder in your local system after point 3 above. 
5. Hit python plt_reg.py in your local terminal.
Enjoy:)
